# Архиватор Хаффмана

Консольный архиватор файлов, основанный на алгоритме кодирования Хаффмана.

## Сборка

make

## Использование

Сжатие файла:

./huff c input.txt output.huff

Распаковка файла:

./huff d archive.huff output.txt

## Алгоритм

Программа использует кодирование Хаффмана для сжатия файлов без потерь.

Шаги:

1. Подсчет частот байтов
2. Построение дерева Хаффмана с использованием очереди с приоритетами
3. Генерация префиксных кодов
4. Запись сжатого битового потока

## Структура проекта

main.c — интерфейс командной строки
huffman_tree.c — дерево Хаффмана и очередь с приоритетами
huffman_codec.c — логика сжатия/распаковки
bitio.c — битовый ввод/вывод

## Система сборки

Makefile (gcc)
