# Huffman Archiver

Консольный архиватор файлов на основе алгоритма **кодирования Хаффмана**.

## Сборка

```bash
$ make
```

## Использование

### Сжатие файла

```bash
$ ./huff c input.txt archive.huff
```

### Распаковка файла

```bash
$ ./huff d archive.huff output.txt
```

## Алгоритм

Программа использует **кодирование Хаффмана** для сжатия файлов без потерь.

Основные шаги:

1. Подсчёт частот байтов во входном файле
2. Построение дерева Хаффмана (priority queue)
3. Генерация префиксных кодов
4. Запись сжатого битового потока

## Структура проекта

```
src/
  main.c            CLI интерфейс
  huffman_tree.c    дерево Хаффмана
  huffman_codec.c   логика сжатия
  bitio.c           битовый ввод/вывод

include/
  *.h               заголовочные файлы
```
